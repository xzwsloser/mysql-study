# 数据库的相关概念  
数据库: 存储数据的仓库,数据是有组织的进行存储(简称DB)  
数据库管理系统: 操纵和管理数据库的大型软件(DBMS)  
SQL:操纵关系型数据库的编程语言,定义了一套操作关系型数据库的统一标准(SQL)   
# MySQL的启动和停止  
1. win+r 输入services.msc 进入服务,启动mysql数据库  
2. 命令行输入net start mysql80 启动mysql  输入net stop mysql80停止数据库  
# MySQL的客户端连接  
1. 方式一: 利用MySQL提供的客户端命令行工具连接  
2. 方式二: 利用windows的命令行 : 命令: mysql [-h 127.0.0.1][-P 3306] -u root -p  
## 数据模型  
1. DBMS用于操纵数据库,而数据库中有多张表结构,表才是最终存储数据的  
2. 关系型数据库(RDBMS):建立在关系模型基础上,有多张相互连接(不同的表之间相互联系)的二维表组成的数据库   
3. 特点:
   1. 使用表存储数据,格式统一,便于维护  
   2. 使用SQL语言操纵,标准统一,使用方便  
4. 数据模型: 使用客户端连接服务器,使用DBMS操纵多个数据库,每个数据库中又可以存储多张表
